# BrainBoost Knowledge Hub

BrainBoost Knowledge Hub is an educational platform dedicated to providing quality tuition classes, academic support, and structured learning resources for students.

## About

This website serves as the official online presence of BrainBoost Knowledge Hub. It provides information about courses, learning programs, and student support services designed to help learners achieve academic excellence.

## Features

- Information about tuition classes
- Course details and learning programs
- Student admission information
- Contact and enquiry options
- WhatsApp communication support

## Purpose

The goal of BrainBoost Knowledge Hub is to empower students through personalized guidance, concept clarity, and consistent academic mentorship.

## Website Access

The website is hosted using GitHub Pages and can be accessed online once published.
